Screen Shader - Common Issues

Not working on the new tab page?

Unfortunately chrome extension permissions prevent Screen Shader from shading the new tab, 😢 however, installing the Screen Shader Dark Theme or any other dark theme from the chrome web store will change the new tab to a color that isn't bright white! 😎

White flash?

Sometimes when switching tabs chrome might flash bright white as it loads the page. 😕 Screen Shader tries to shade the page as soon as possible, but often the browser is too slow and the result is that the screen flashes white between changing tabs. This can be fixed by installing the Screen Shader Dark Theme which changes that white color to black. ⛱

Not working in incognito?

Chrome extensions are disabled by default in incognito mode. If you'd like Screen Shader or any other extension to work you can go to the chrome://extensions/ page, click on Screen Shader and check "Allow in incognito."

Screen Shader slowing down chrome or pixelating part of the page?

Click on the Screen Shader icon, navigate to the "Settings" page and change "Color Blending Algorithm" to "Opacity". This shading algorithm is the fastest to compute and will use less processing power than the others at the expense of looking worse. ⏳

Can't scroll?

The scrolling issue on pdfs and other imbedded items was a bug in chrome that was fixed. Update chrome to version 72 or above.
